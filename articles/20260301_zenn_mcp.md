# Claude Code から Zenn 記事を自動投稿できるようにした話

Claude Code を使い始めて数日、ふと気づいた。

「プロジェクトの記録を毎回 Zenn に手で書くの、めんどくさくない？」

そこで調べてみると、**MCP（Model Context Protocol）** というしくみを使えば、Claude Code から直接 Zenn に記事を投稿できることがわかった。早速やってみた記録がこれだ。

---

## MCP とは何か

MCP（Model Context Protocol）は、Claude Code が外部ツールやサービスと連携するためのプロトコル。

たとえば Notion、GitHub、Zenn などのサービスを MCP サーバーとして接続すると、Claude Code の中から「Notion にメモを追加して」とか「Zenn に記事を作って」が言えるようになる。

今回は **Zenn MCP**（`zenn-content-mcp`）を導入した。

---

## 環境

- OS: Ubuntu（WSL2 on Windows）
- Claude Code: CLI 版
- Node.js: v24.14.0（nvm でインストール）

---

## セットアップ手順

### 1. Node.js のインストール

```bash
# nvm をインストール
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash
source ~/.nvm/nvm.sh

# LTS 版をインストール
nvm install --lts
```

### 2. zenn-content-mcp をクローン・ビルド

```bash
cd ~/workspace/claudecode/00006_zenn_mcp_20260301
git clone https://github.com/inoue2302/zenn-content-mcp.git
cd zenn-content-mcp
npm install
npm run build
```

### 3. Zenn コンテンツリポジトリをクローン

Zenn と GitHub 連携済みのリポジトリをローカルに持ってくる。

```bash
gh repo clone JapanNomu/zenn-content ~/workspace/zenn-content
```

### 4. .mcp.json を作成

Claude Code のプロジェクトフォルダに `.mcp.json` を置く。

```json
{
  "mcpServers": {
    "zenn-content": {
      "command": "/home/japannomu/.nvm/versions/node/v24.14.0/bin/node",
      "args": [
        "/home/japannomu/workspace/claudecode/00006_zenn_mcp_20260301/zenn-content-mcp/dist/index.js"
      ],
      "env": {
        "ZENN_PROJECT_DIR": "/home/japannomu/workspace/zenn-content"
      }
    }
  }
}
```

ポイントは `ZENN_PROJECT_DIR` で Zenn リポジトリのパスを指定すること。

### 5. Claude Code を再起動

```bash
claude --continue
```

起動時にこんな画面が出る。

```
New MCP server found in .mcp.json: zenn-content
1. Use this and all future MCP servers in this project
2. Use this MCP server
3. Continue without using this MCP server
```

**1 を選択する**（今後起動のたびに聞かれなくなる）。

### 6. 接続確認

Claude Code の中で `/mcp` コマンドを実行すると、こんな表示になれば成功。

```
Project MCPs (/home/japannomu/workspace/claudecode/.mcp.json)
> zenn-content • ✓ connected
```

---

## 使ってみる

接続できたら、あとは普通に会話するだけ。

```
Zenn の記事一覧を見せて
```

```
新しい Zenn 記事を作って
```

これだけで MCP 経由で記事の作成・更新が動く。手で Markdown ファイルを作る必要がなくなる。

---

## まとめ

| ステップ | 内容 |
|---------|------|
| MCP サーバー | `zenn-content-mcp`（OSS） |
| 設定ファイル | `.mcp.json`（プロジェクトフォルダに置く） |
| 再起動 | `claude --continue` で MCP を認識 |
| 確認 | `/mcp` で `✓ connected` を確認 |

Claude Code を使い始めたばかりでも、MCP を入れるだけで Zenn 投稿の自動化ができた。

プロジェクトを作るたびに記事に残していけると、後から振り返りやすくなる。これは続けていきたい。
